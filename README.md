### HANGMAN ###

### About ###

>Hangman game.

-maintain data after server/client reload
-send SMS
-P2P browser communication
-global statistics page

- HTML5
- JavaScript
- NodeJS
- Bootstrap
- SMS NEXMO API

### Set up ###
```sh
$ cd  src
$ npm install
$ node server.js
```
* point a browser (chrome) to http://localhost:8080
